## 🛠️ Tech Stack

- **Frontend:** React, Tailwind CSS
- **Backend:** Node.js + Express
- **Database:** MongoDB 
- **APIs:** Google Maps, Socket-io

---

## 🚀 Features

- Real-time cab booking
- Fare estimation
- Google Maps Api
- Live tracking
- Secure payment
- cors
- CURD operations
